Restaurant Food Delivery — Microservices on AWS EKS

Spring Boot microservices deployed to AWS EKS with GitOps. Angular frontend, API Gateway/Service Discovery, and automated CI/CD.

Tech

Spring Boot Eureka Angular MySQL (RDS) MongoDB Atlas Docker Jenkins ArgoCD AWS EKS + ALB

What I built
  • Services: FoodCatalog, User, Restaurant, Orders with REST APIs & JPA.
  • Discovery & config via Eureka; API gateway routing.
  • CI/CD: Jenkins → Docker images → ArgoCD GitOps → EKS.
  • Datastores: MySQL (RDS) for core entities; MongoDB Atlas for Orders.
  • Quality gates with SonarQube/JaCoCo; smoke tests via Postman.
