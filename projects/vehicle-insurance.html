Vehicle Insurance Management — Java · JSP · Spring Boot

Internship project (team). I led backend; JSP-based UI. Customers apply for policies, admins/field officers validate, policy admins approve; helpdesk tickets for all roles.

Tech

Java Spring Boot JSP MySQL

Core features
  • Roles: Customer, Admin, Field Officer, Policy Admin.
  • Customer: registration, profile, submit proofs, track approval, add reviews.
  • Admin: approve officer/policy-admin logins, route assignments, handle help requests.
  • Field Officer / Policy Admin: validate proofs, approve/reject, respond to tickets.
